Introduction

Visual Studio Code (VSCode) is a powerful, lightweight code editor developed by Microsoft. It supports a wide range of programming languages and comes packed with features such as IntelliSense, debugging, Git integration, and extensions for custom workflows. Its clean interface and rich functionality make it one of the most popular code editors among developers.

This documentation page covers the most essential keyboard shortcuts in VSCode to help you navigate, edit, and manage your projects more efficiently. Mastering these shortcuts can help you streamline your workflow and focus more on writing code.

General Shortcuts

These shortcuts are the basic actions that are common across the application.

Ctrl + Shift + P
  • Open Command Palette
Ctrl + P
  • Quick Open
Ctrl + N
  • Open New File
Ctrl + S
  • Save File
Ctrl + W
  • Close File

Navigation Shortcuts

Efficient navigation within your workspace is key to productivity. Below are the main shortcuts for navigating files and symbols.

Ctrl + P
  • Go to File
Ctrl + G
  • Go to Line
F12
  • Go to Definition
Ctrl + Shift + O
  • Go to Symbol
Shift + F12
  • Show References
Search and Replace Shortcuts

Search and replace is a key part of the development process. Below are the shortcuts for finding and replacing content in your project.

Ctrl + F
  • Find
Ctrl + H
  • Replace
Ctrl + Shift + F
  • Find in Files
Ctrl + Shift + H
  • Replace in Files
Ctrl + Shift + F
  • Toggle Search/Replace Panel
